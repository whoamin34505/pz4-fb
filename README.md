# Avion Furniture

Лендинг мебельного бренда Avion

## Описание

Статическая HTML-страница интернет-магазина мебели с фиксированной навигацией, мобильным меню, баннером, каталогом товаров, преимуществами бренда, формой подписки и информацией о компании.

## Технологии

* HTML5
* CSS3 с использованием БЭМ методологии
* Google Fonts (Montserrat)
* Flexbox — навигация и макеты реализованы с использованием flex-контейнеров
* CSS Grid — созданы две разные сетки (равномерная для товаров и masonry для популярных товаров)
* Медиазапросы — реализован Desktop First подход с breakpoints (1199px, 991px, 767px, 575px, 399px)
* JavaScript — управление мобильным меню и кнопкой «Наверх»

## Особенности

* Навигация с якорными ссылками и фиксированной шапкой
* Мобильное off-canvas меню с оверлеем и кнопкой-гамбургером
* Семантическая разметка
* CSS стили в style.css с применением БЭМ (Block Element Modifier) для организации классов
* Hover/active эффекты на интерактивные элементы
* **Flexbox-раскладка** — навигация, шапка и основные компоненты используют flex-контейнеры для гибкого размещения
* **Две сетки Grid**:
  - Равномерная сетка 4 колонки для "New ceramics" и начальной части "Popular products"
  - Masonry макет 3 колонки для "Popular products" с асимметричным распределением элементов
* **Адаптивность** — страница корректно отображается на всех типах устройств (мобильные, планшеты, десктопы)
* **Desktop First подход** — стили написаны для десктопа, затем переопределяются для меньших экранов, реализовано 5 breakpoints (399px, 575px, 767px, 991px, 1199px)
* **Типографика** — на мобильных экранах минимальный размер текста 17px
* **Кнопка «Наверх»** — появляется при прокрутке и плавно скроллит страницу в начало

## Структура проекта

* `index.html` - главная страница
* `styles/style.css` - стили с БЭМ классами
* `styles/mobile.css` - адаптивные стили и мобильные правки
* `scripts/menu.js` - логика мобильного меню и кнопки «Наверх»
* `assets/img/` - изображения
* `assets/icons/` - иконки
